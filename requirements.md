# DDL 倒计时 App v1.0 需求文档

## 概述

一款面向大学生的 DDL（截止日期）倒计时管理应用，帮助学生在手机上清晰追踪各项作业和任务的截止时间。

- **目标平台**：Android（优先适配）
- **技术栈**：React Native + Expo + TypeScript
- **风格**：简洁，适合学生使用

---

## 功能需求

### 1. 首页 — DDL 列表

- 展示所有 DDL，按截止日期**从近到远**排列（最紧迫的排在最前面）
- 每条 DDL 显示：
  - DDL 名称
  - 剩余时间：
    - 大于等于 1 天 → 显示"X 天"
    - 不足 1 天 → 显示"X 小时 X 分钟"
- 支持三种状态展示：
  - 进行中（未过期）
  - 已完成（用户手动标记）
  - 已过期（超过截止日期且未完成，自动标记）
- 已过期的 DDL 保留在列表中，显示"已过期"标识

### 2. 添加 DDL

- 输入 DDL 名称
- 选择截止日期和时间（精确到分钟）
- 添加后自动返回首页，列表刷新

### 3. 状态操作

- 手动标记已完成 → 从活跃列表中移出（或视觉区分）
- 超过截止日期 → 自动标记为"已过期"
- 已完成和已过期的 DDL 仍展示在列表中

### 4. 数据持久化

- 所有 DDL 数据存储在手机本地（AsyncStorage）
- 关闭 App 后数据不丢失
- 重新打开 App 时自动加载已保存的数据

---

## UI 设计要点

- 首页：DDL 列表 + 右下角添加按钮（FAB）
- 添加页：名称输入框 + 日期时间选择器 + 确认按钮
- 触摸区域最小 44x44pt
- 支持 Android 系统

---

## 状态处理

| 场景 | 处理方式 |
|------|----------|
| 首次打开（无数据） | 显示空状态提示 |
| 数据加载中 | 显示加载状态 |
| 数据加载失败 | 显示错误提示 |
| 列表有数据 | 正常展示 |
| 列表全部完成/过期 | 正常展示 |
| 添加成功 | 返回首页，列表自动更新 |
