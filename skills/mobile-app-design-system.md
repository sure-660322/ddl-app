# Mobile App Development Skills

## React Native Expo Development
- 使用 React Native + Expo 构建跨平台移动应用
- 所有包管理优先使用 expo 官方工具
- 使用 Expo Router 做页面导航
- 使用 Zustand 管理状态
- 使用 AsyncStorage 做本地数据持久化
- 列表渲染使用 FlatList，注意性能优化
- 严格遵循 React Native 最佳实践
- 使用 TypeScript

## Mobile App Design System
- 遵循 iOS HIG 和 Android Material 3 设计规范
- 触摸按钮最小尺寸 44x44pt
- 重要操作放在屏幕下半部分拇指容易触达的区域
- 颜色、字体、间距保持统一的设计风格
- 支持深色模式
- 避免低效的列表渲染方式
